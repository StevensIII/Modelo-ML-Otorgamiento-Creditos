Instrucciones para correr la aplicación:

1. Ejecuta el archivo "main.py".

2. Configura la base de datos:
   - Abre SQL Server Management Studio (SSMS).
   - Conéctate a tu servidor de SQL Server.
   - Crea una nueva base de datos (o usa una existente).
   - Abre el archivo "crear_tablas.sql" y ejecuta el script.
   - Ejecuta el script (presiona F5 o haz clic en "Ejecutar").

3. Configura la conexión:
   - Abre el archivo "config.py" y modifica los valores de "DATABASE_CONFIG" con los detalles de tu servidor.

4. ¡Listo! La aplicación está lista para usarse.

5. El articulo cientifico con los detalles del proceso lo encontraras como "Articulo Cientifico Soft ML Aprobaciones - IEEE - Esp.pdf".

6. En el archivo "Licencia.txt" encontraras la clave de licencia para ejecutar el aplicativo.
