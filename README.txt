Instrucciones para usar la aplicación:

1. Extrae el contenido del archivo ZIP en una carpeta.

2. Ejecuta el archivo "main.exe".

3. Configura la base de datos:
   - Abre SQL Server Management Studio (SSMS).
   - Conéctate a tu servidor de SQL Server.
   - Crea una nueva base de datos (o usa una existente).
   - Abre el archivo "crear_tablas.sql" y ejecuta el script.
   - Ejecuta el script (presiona F5 o haz clic en "Ejecutar").

4. Configura la conexión:
   - Abre el archivo "config.py" y modifica los valores de "DATABASE_CONFIG" con los detalles de tu servidor.

5. ¡Listo! La aplicación está lista para usarse.
